# 用户使用手册

## 概述

人机协同A股智能投资决策系统是一个基于AI的股票预测和投资辅助工具，帮助个人投资者做出更明智的投资决策。

---

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库

首次使用时，数据库会自动创建。你也可以手动初始化：

```python
from data.database import DatabaseManager

db = DatabaseManager()
db.create_tables()
```

### 3. 更新数据

首次使用需要先更新数据：

1. 启动应用：`streamlit run app.py`
2. 在侧边栏选择"数据更新"
3. 点击"更新板块数据"按钮
4. 点击"更新技术指标"按钮

### 4. 训练模型（可选）

如果需要使用预测功能，需要先训练模型：

```bash
python prediction/trainer.py
```

## 主要功能

### 1. 首页 - 板块总览

查看A股各板块及龙头股信息：

- **板块选择**：选择感兴趣的板块（行业板块或概念板块）
- **龙头股列表**：查看该板块的龙头股排名
- **股票信息**：查看股票代码、名称、市值、涨跌幅等

#### 使用步骤：

1. 在首页选择板块类型（行业/概念）
2. 从下拉菜单中选择具体板块
3. 查看该板块的龙头股列表
4. 点击股票名称查看详情

---

### 2. 股票详情页

查看单个股票的详细信息：

- **基本信息**：股票名称、代码、所属板块、市值、市盈率、市净率
- **K线图**：查看历史价格走势
- **成交量图**：查看成交量变化
- **技术指标**：切换查看MACD、RSI、KDJ、BOLL等技术指标
- **预测建议**：查看短中长期预测和综合建议

#### 使用步骤：

1. 从首页点击股票名称进入详情页
2. 选择时间范围查看历史数据
3. 切换技术指标进行分析
4. 查看预测建议和置信度

---

### 3. 数据更新

更新系统数据：

- **板块数据**：更新板块列表和龙头股信息
- **技术指标**：为所有股票计算技术指标

#### 更新频率建议：

- 板块数据：每周更新一次
- 技术指标：每个交易日收盘后更新

---

### 4. 历史回顾（功能开发中）

查看历史预测准确率和表现：

- 准确率统计
- 准确率趋势图
- 历史预测记录

## 预测功能说明

### 三周期预测

系统提供三个时间维度的预测：

| 周期 | 时间范围 | 主要依据 | 权重 |
|------|----------|----------|------|
| 短期 | 1-5天 | 技术指标、量价关系 | 30% |
| 中期 | 1-3个月 | 趋势、均线系统 | 40% |
| 长期 | 3个月以上 | 价值分析、行业周期 | 30% |

### 预测建议

系统会给出三种建议：

- **买入（红色）**：建议考虑买入
- **卖出（绿色）**：建议考虑卖出
- **持有（灰色）**：建议继续持有

### 置信度

预测结果附带置信度（0-1），表示模型对该预测的信心程度。

### 免责声明

> ⚠️ 重要提示：系统预测仅供参考，不构成投资建议。投资有风险，入市需谨慎。请根据自身风险承受能力做出投资决策。

## 常见问题

### Q1: 为什么有些股票没有预测结果？

A: 可能的原因：
1. 数据不足（需要至少60天历史数据）
2. 模型尚未训练
3. 技术指标计算失败

### Q2: 如何提高预测准确率？

A:
1. 确保数据完整且最新
2. 定期重新训练模型
3. 参考多个时间周期的预测结果
4. 结合基本面分析

### Q3: 系统支持哪些股票？

A: 目前支持A股市场（上海证券交易所和深圳证券交易所）的主板、中小板和创业板股票。

### Q4: 数据从哪里获取？

A: 数据来源于akshare（免费开源的财经数据接口）。

### Q5: 如何备份数据？

A: 数据库文件位于 `data/stock_data.db`，直接复制该文件即可备份。
